HTML : It stands for Hyper Text Markup Language 

Hypertext means Link
Markup means structure

The extension of Html is .htm or .html

It is inroduced by Tim-Berners-Lee in the year of 1993

The current version of HTML is 5

It is used to create Layout for  web pages or website by our own.


T : tag
E : Element
A : Attribute

The Structure of Html is:
<!DOCTYPE html>
<html>
    <head></head>
    <body>

    </body>
</html>
